# Qt-Theremin
Proyecto del Theremin adaptado a QT5

Dispositivos necesarios para su uso

-Leap Motion
-Teclado
-Raton

Requiere el uso de las siguientes librerias y aplicaciones

-Visual Studio 2012
-assimp 3.0.1270
-leap motion 2.2.4
-glut 7.3.6
-qt 5.5.0
-qt Visual Studio Add-in 1.2.2
-fmod studio 1.06
